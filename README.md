## ⚙️ How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/smitkachhadiya/DSA.git
   ```
2. Navigate to the folder:

   ```bash
   cd DSA
   ```
3. Run the code (example for Java):

   ```bash
   javac Arrays/TwoSum.java
   java Arrays.TwoSum
   ```
